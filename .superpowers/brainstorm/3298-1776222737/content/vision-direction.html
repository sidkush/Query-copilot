What does "voice + agent + chart" FEEL like?

Three fundamentally different design directions. Pick the one that excites you most — we can hybridize, but the dominant pattern shapes everything below it.

A

Voice-First Studio

Holographic. Cinematic. Iron Man HUD energy.

"Show me revenue trends by region…"
▸Voice is the primary input. Mic always-listening. UI collapses to a single glowing orb until you speak.
▸Chart fills the screen. No rails, no inspector, no menus. Edit by talking. "Make it stacked." "Color by churn risk."
▸Agent narrates back. "Done. Notice the spike in March — want me to drill into it?"
▸Inspector + data rail are glyphs. Float in only when needed, fade out instantly.
▸Wow ceiling: maximum. Best demo. Best for novice analysts and execs.
B

Conversational Composer

ChatGPT for charts. Voice + chat + live canvas.

"trend by region"
picking line chart…
grouping by region…
"add forecast"
Revenue by Region
▸Split view, always. Chat panel left (voice transcribes here), live chart canvas right.
▸Agent steps visible. See thinking, tool calls, decisions stream as the chart materializes — Claude Code style.
▸Voice OR text, your choice. Push-to-talk or always-on. Hot-swappable.
▸Click the chart to refine inline. Direct manipulation as fallback when voice can't say it precisely.
▸Wow ceiling: high. Best for analysts who want transparency into what the agent is doing.
C

Power Editor + Voice

Tableau-class direct manipulation. Voice as accelerator.

▸Three-pane Tableau-class editor. Data rail + canvas + inspector. Marks card with draggable pills.
▸Voice is a power-user accelerator. "Color by region" → pill jumps to Color slot. Visual feedback.
▸Every voice command has a visible UI equivalent. Users learn the editor by watching voice drive it.
▸Best for serious analysts. Maximum flexibility, minimum magic.
▸Wow ceiling: medium-high. Looks more like Tableau than a movie. Familiarity wins enterprise sales.

My recommendation: A hybrid — B as the default mode (conversational composer is the broad-audience hero), with A as a "Stage" mode for demos / execs / focus work, and C as a "Pro" mode for analysts who want full Tableau-class control. All three share the same underlying IR + agent + voice pipeline. You build one architecture, three faces.
