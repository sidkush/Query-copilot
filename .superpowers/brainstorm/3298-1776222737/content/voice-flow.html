Hybrid (User Picks)

All three modes ship. User toggles per session or workspace.

Default = Push-to-Talk (safe, predictable)
Toggle to Wake Word for hands-free demos and Stage Mode
Toggle to Hot Mic for power-user dictation sessions
Per-workspace setting + global override
Stage Mode auto-switches to wake-word for cinematic demo flow
Pros User autonomy · Right tool per context · Works for everyone · Stage Mode magic + analyst safety
Cons Most engineering · Three flows to test · Settings UI complexity

My recommendation: D (Hybrid). Default is Push-to-Talk for safety + open-office support. Stage Mode auto-flips to Wake Word ("Hey Ask") for the cinematic demo flow. Power users can opt into Hot Mic. Engineering cost is real (3 flows), but voice + agent IS the differentiator and a single flow won't fit every customer. Custom wake words per workspace (e.g., "Hey Stark" for an Iron Man brand) become a viral wow factor.
