My recommendation: Ship all six. They're not redundant — they're modes of the same underlying tile-on-grid system. The same dashboard data can be viewed as Executive Briefing OR Analyst Workbench OR Pitch Deck depending on who's looking. Mode is a one-click toggle. The PresentationEngine is already in your codebase — that's mode 5. Mode 1 is a tile layout preset. Mode 2 is a tile layout preset. Mode 3 needs websocket refresh + alert pills.
