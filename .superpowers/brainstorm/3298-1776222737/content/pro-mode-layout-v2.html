My recommendation (unchanged): Layout 1 (Tableau Classic) + Cmd+. Focus Mode from Layout 3. Marks card always visible is Tableau's biggest leverage. Layout 2 hides encoding under a tab — too much friction for analysts. Layout 3's floating Marks is cool but discoverability is worse for newcomers. Layout 1 + Focus Mode (collapse both rails for distraction-free editing) gives you Tableau's power AND Hex's modern feel.

This is the LIGHT theme. Dark theme will use the same layout with charcoal panels (#1a1a1f canvas, #2a2a30 rails, #ffffff text). Pick a layout — next question is the dashboard, which is where the real Tableau gap lives.
