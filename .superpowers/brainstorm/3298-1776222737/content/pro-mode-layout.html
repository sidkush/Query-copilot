Hex-Style Floating Marks

Both rails collapsible. Marks card is a draggable floating panel.

Both side rails collapse to 0px on demand (Cmd+\ + Cmd+B)
Marks card is a FLOATING glass panel, draggable anywhere on canvas
Right rail simplified: Style + Annotations only (no encoding — that's in Marks)
Pro Mode → Stage Mode = collapse rails + slide Marks panel offscreen
Most flexible · most modern · biggest learning curve
ProsMost canvas when collapsed · Smooth Pro→Stage transition · Modern feel · Marks is draggable hero · Customizable per workspace
ConsFloating panel is fiddly to find · Most engineering · Discoverability lower for newcomers

My recommendation: 1 (Tableau Classic). The Marks card visible at all times is the single biggest leverage point in Tableau's UX — every encoding, every channel, immediately addressable, no tab switching. Analysts in Pro Mode want to SEE the encoding tray while editing the inspector. Add the collapsible-rails behavior from Layout 3 as a "Focus mode" toggle (Cmd+. collapses both rails for distraction-free) — that gives you the modern feel without losing the always-visible Marks card. Layout 2 is too compressed for serious analysts who need both encoding and style visible simultaneously.

All three preserve the bottom dock (mic + chat input) from Default Mode — voice + agent + chart still work everywhere. Pick a layout, or describe a different arrangement.
