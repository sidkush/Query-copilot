My Recommendation
Build order: SP-1 → SP-2 → SP-3 (parallel with SP-4) → SP-5 → SP-6

SP-1 (Shell) is 2-3 days of foundation that unblocks everything. SP-2 (Agent Panel) is the wow moment — the conversational composer from wireframe 5 is what makes AskDB different from every other BI tool. SP-3 and SP-4 can run in parallel (text tiles don't depend on theme, theme doesn't depend on text tiles). SP-5 (Voice) needs agent panel + status bar, so it comes after. SP-6 is the final polish pass when all pieces exist.

Total estimate: ~18-24 days across 6 sub-projects. Each sub-project gets its own brainstorm → spec → plan → build cycle. We spec SP-1 now, build it, then spec SP-2, and so on.
